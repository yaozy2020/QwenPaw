本文将为您列举阿里云CLI调用OpenAPI时可用的命令行选项及其作用，帮助您修改命令默认行为或开启部分功能。  

## **选项格式**
命令行选项`options`可用于OpenAPI通用命令之后，格式如下：

```
HELPCODEESCAPE-shell
aliyun <Command> <SubCommand> --options [optionParams]
```

* 选项与参数之间通常使用空格隔开。若选项包含子字段`subfield`，则通常以`subfield=value`形式传入。更多信息，请参见[参数格式](https://help.aliyun.com/document_detail/110340.html)。
