## Настройка инструментов

Навыки (Skills) определяют, _как_ работают инструменты. Этот файл предназначен для _ваших_ особенностей — того, что уникально для вашей настройки.

### Виртуальное окружение Python

**Важно:** Все команды Python выполняются в виртуальном окружении. Сначала активируйте его:

```bash
source /var/apps/com.dustinky.qwenpaw/home/venv/bin/activate
```

После активации используйте `python3` или `pip3` напрямую. Виртуальное окружение уже настроено со всеми необходимыми зависимостями.

### Среда Node.js

**Важно:** Все команды Node.js выполняются в виртуальном окружении. Сначала активируйте его:

```bash
export PATH=/var/apps/nodejs_v22/target/bin:$PATH
```

После активации используйте `node` или `npm` напрямую. Виртуальное окружение уже настроено со всеми необходимыми зависимостями.

### Что здесь записывать

Добавляйте все, что помогает вам выполнять свою работу. Это ваша шпаргалка.

Например:

- Хосты SSH и псевдонимы (aliases).
- Другие настройки пользователя при выполнении навыков.
